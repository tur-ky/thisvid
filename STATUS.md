### Critique (Defensive Audit)
- **Rule 7 (IPC Safety):** `isTauri()` checks `window.__TAURI_INTERNALS__` before every import. Tauri APIs are dynamically imported inside `tauriInvoke()` and `tauriListen()` wrappers — never at module load time. Renders gracefully in Next.js static build.
- **Rule 2 (Stale Closures):** `useEffect` returns a cleanup that calls all unlisten fns; `mounted` flag prevents callbacks after unmount.
- **Rule 1 (Async Blindness):** Task status set to `fetching_metadata` immediately on submit; UI reflects pending state.
- **Rule 4 (Idempotency):** `addTask` rejects duplicate URLs with active statuses — returns `null`, no-ops silently.
- **Rule 5 (Loud Failures):** Every `catch` block calls `updateTaskError()`. One `console.warn` is used for a non-fatal cancel IPC failure (intentional — task already marked cancelled in store).
- **Rule 6 (No Magic Variables):** All IPC command names, event names, port, truncation length imported from `@thisvid/core/constants`.
- No discrepancies found in revision phase. Proceeding to Step 4.
